# Car-Race
A simple car racing game in python

# Usage
``` python
from Race import Race
myRace = Race(length of race, speed of cars)
myRace.addCar(name of car)
myRace.start()
```
# Example

``` python
from Race import Race
formula1 = Race(100, 1)
formula1.addCar('McLaren')
formula1.addCar('Renault')
formula1.start()
```
